# Problem: Find All Duplicates in an Array

## Problem Statement

Given an array of integers where 1 ≤ a[i] ≤ n (n = size of array), some elements appear twice and others once.

Find all the elements that appear twice.

## Example 1:

```text
Input: [4,3,2,7,8,2,3,1]
Output: [2,3]
```

## Example 2:

```text
Input: [1,1,2]
Output: [1]
```
